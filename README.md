ELIXIR ERP is a complete Enterprise Resource Planning (ERP) system UI/UX design created to simulate a real-world industrial ERP application.

This repository contains the full design flow starting from the Login Page to all major functional modules. The design focuses on:

Clean enterprise layout

Consistent Intel Blue theme

Structured data representation

Logical module flow

User-friendly navigation

Professional corporate appearance

This project demonstrates ERP workflow understanding and enterprise-level interface planning.
